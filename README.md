# KaryGo
A modern mobile application that connects job seekers with job providers, offering freelance opportunities along with smart career guidance, productivity tools, and AI-powered recommendations.

## Features
- ***Freelance Marketplace***: Connects freelancers with clients for short-term and long-term projects.
- ***AI-Powered Recommendations***: Suggests jobs, skills, and opportunities based on user behavior and profile.
- ***Smart Career Guidance***: Personalized career insights and growth suggestions.
- ***Productivity Tools***: Tools to manage tasks, deadlines, and work efficiency.
- ***Advanced Job Search & Filters***: Easily find jobs based on category, experience level, and budget.
- ***Save & Like Jobs***: Bookmark jobs for later viewing.
- ***Real-time Notifications***: Stay updated with job postings, proposals, and messages.

## Tech Stack
#### Frontend: React Native / Javascript
#### Backend: Django REST Framework
#### Database: PostgreSQL
#### AI Integration: Langchain
